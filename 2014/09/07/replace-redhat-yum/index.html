替换redhat的yum安装源

删除 redhat 原有的 yum 源

# rpm -aq | grep yum|xargs rpm -e --nodeps

下载新的 yum 安装包

  • 此处使用 CentOS 的 yum 源

安装 yum 软件包

  • 注意:最后两个安装包相互依赖,要放在一起同时安装,否则安装失败.

更改 yum 源

  • 使用网易的 CentOS 镜像源
  • 编辑文件,把$releasever全部替换为版本号“6”,
  • 或者直接把以下内容拷贝到CentOS6-Base-163.repo文件中(已修改)

清理 yum 缓存

# yum clean all
# yum makecache       //将服务器上的软件包信息缓存到本地,以提高搜索安装软件的速度
